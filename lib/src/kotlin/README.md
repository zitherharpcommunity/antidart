# Kotlin vs. Dart

## Classes
| Kotlin | Dart |
| - | - |
| `class` | `class` |
| `abstract class` | `abstract class` |
| `annotation class` | `final class` |
| `data class` | `final class/record` |
| `enum class` | `enum` |
| `interface` | `abstract interface class` |
| `object/companion object` | `sealed class` |
| `open class` | `external class` |
